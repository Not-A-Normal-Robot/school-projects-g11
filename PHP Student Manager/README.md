# PHP Student Manager
## Setup
1. Install PHP
2. Install Composer
3. `composer install`
4. Create a `.env` file with the following format: ```
DB_HOST=(mysql host, usually `localhost`)
DB_USERNAME=(mysql username)
DB_PASSWORD=(mysql password)
DB_NAME=(mysql database name, usually `PhpDbSiswa`)
```
5. `php -S localhost:8000`
6. Open `localhost:8000/index.php` in your browser
